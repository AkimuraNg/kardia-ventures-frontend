# Kardia Ventures

This website was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Deployment

### Requirements

* node
* npm 

### Production build
* Install React

    ```shell
    $ npm install react
    ```

* Static Server
            
    ```shell
    $ npm install -g serve
    ```

* Build

    ```shell
    $ npx serve -s build
    ```

* Serve file is stored in `build` directory with HTTP server.
